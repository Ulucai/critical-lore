# Companhia das Feras

!!! info ""
    "Não é nada pessoal, são apenas negócios."
# Descrição
A Companhia das Feras é o nome que se dá ao pacto entre quatro sub-organizações independentes: [Medusa de Platina](3a5d9f85-bc19-4015-8e26-6b9f47badaf7.md); [Hidra de Ouro](b8f43174-a780-4eb3-b63e-12e3e5bee304.md); [Manticora de Prata](b2b67d9f-b91d-4b0e-94a6-2936cf94dbad.md) e [Cerberus de Bronze](bdf09cf4-387b-4c75-96bb-5dc6b0041d68.md).

Cada uma dessas organizações possui várias fachadas para operar seus negócios. Sendo a mais utilizada a de uma companhia mercantil espalhada por toda Pheros.
Como cada uma das sub-organizações possui uma moeda com sua monstruosidade cunhada como brasão, internamente elas são frequentemente chamadas por moedas de platina, ouro, prata e bronze, respectivamente.
<br>

### Localidade

Espalhada por toda Pheros, com bases em grandes capitais e centros econômicos: Garmareia, Porto Tempestade, Fortuna e Capital Dourada, são alguns exemplos.

### Regras
A Companhia das Feras possui um pacto que respeita às seguintes regras:

• Uma moeda não possui valor maior que as outras;

• Autonomia de atividades das moedas;

• Pacto de não-interferência e não-concorrência;

• Não existem favores, apenas negócios. Um trato devido é cobrado com juros;

• O conflito externo de uma moeda não deve ser um fardo para as outras. Porém, se há perigo de uma moeda cair, o conflito se torna da Companhia e a moeda em questão será punida após o conflito.
<br>

### Membros e Hierarquia
Cada uma das moedas possui uma especialidade e um modo de agir, porém sempre obedecendo o pacto instituído entre elas.

Cada subdivisão possui um líder, chamado de Mestre das Feras ou **Beastmaster**, os Beastmasters apenas encontram-se em conselho durante situações emergenciais ou quando há algum “peixe grande” para ser fisgado. Crises internas ou pequenos conflitos costumam ser resolvidos dentro de suas próprias moedas.

As moedas, com seus respectivos Beastmasters, possuem autonomia em suas atividades, porém não é incomum que membros de moedas diferentes colaborem para atingir algum objetivo comum.

Quando os membros precisam se identificar, eles mostram uma pequena moeda cunhada com o brasão de uma das monstruosidades que os representam. Essa moeda possui uma aura mágica de transmutação e conjuração. Cada uma delas é vinculada a uma pessoa e, quando fora da posse dessa pessoa vinculada, passa-se por uma moeda falsificada e sem valor comercial, mas o dono original pode usar uma palavra de comando para a reaver ou para mostrar sua verdadeira aparência.
